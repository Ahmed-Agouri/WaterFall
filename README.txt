[WaterFall]
Welcome to [WaterFall]! This is a school project developed using PHP and designed to run on the XAMPP web server.

Project Overview
• lightweight social media application encompassing fundamental functionalities
• Including user authentication, registration, post creation and deletion with associated captions, real-time tracking
of likes, and the implementation of standard features inherent to social media platforms.
• Designed and implemented a robust database architecture, excelling in CRUD operations, to ensure efficient data
management and retrieval.

Prerequisites
Before you get started, make sure you have the following installed on your machine:

XAMPP
Installation
Download and Install XAMPP:

Download XAMPP from https://www.apachefriends.org/index.html.
Follow the installation instructions for your operating system.
Clone or Download this Repository:

Clone this repository using git clone or download the ZIP file and extract it.
Move Project Folder to XAMPP's htdocs:

Locate the htdocs folder within your XAMPP installation directory.
Move the entire project folder into the htdocs folder.
Start XAMPP:

Launch XAMPP and start the Apache server.
Access the Website:

Open your web browser and navigate to http://localhost/[WaterFall].


Configuration
Ensure you have started appache and sql database from XAMPP to run the website smoothly.
